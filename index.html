What is Precifio?

Precifio is a full-stack AI system I designed and built to process natural language notes into structured financial data. It combines GPT, vector embeddings, and an event-driven architecture to handle real-time note editing, automated recurring entries, and multi-user collaboration — all while maintaining consistency and minimizing processing cost.

Built as a real-world system handling asynchronous processing, multi-user workflows, and repeated note transformations reliably.

Core Systems & Capabilities

🔍 Natural Language Query Engine

🎤 Voice-to-Structured Input System

🧠 Engineering Breakdown

Hybrid AI Pipeline

Built a cost-optimized system that prioritizes vector matching and only falls back to GPT when necessary.

  • • Vector-first classification (fast path)
  • • GPT fallback for unmatched segments
  • • Self-learning via embedding storage
  • • Financial gate to avoid unnecessary processing

Queue-Based Processing System

Designed an async pipeline using a pending_notes queue to decouple user actions from AI execution.

  • • Job locking prevents duplicate runs
  • • Idempotent processing
  • • Retry-safe architecture
  • • Handles both manual and auto flows

Edit Detection System

Implemented fingerprint-based diff logic to detect meaningful changes and avoid unnecessary processing or billing.

  • • Canonical text normalization
  • • Dual diff (semantic + raw)
  • • Prevents duplicate charges

Diff-Based Sync Engine

Built a reconciliation system that updates only changed financial entries using stable IDs.

  • • Git-like create/update/delete diff
  • • Stable entry IDs for consistency
  • • Prevents duplication across edits

Auto Notes Engine

Cron-driven system that executes recurring notes and feeds them into the same processing pipeline.

  • • Prevents duplicate daily execution
  • • Permission-based shared execution
  • • Queue-based processing for scalability

Multi-User Collaboration

Designed a shared note system with permission-aware execution and ownership integrity.

  • • Edit permission enforcement
  • • Prevents duplicate execution across users
  • • Maintains single source of truth

🧠 Real-Time Financial Behavior Engine

Beyond simple budgeting, I designed a system that models user financial behavior in real time — combining structured data, time awareness, and intelligent feedback to drive better decisions.

Context-Aware Budget Tracking

Budgets dynamically evaluate real financial activity by combining manual and automated entries with strict filtering.

Time-Aware Budget Logic

  • • Aligns with real-world financial timelines

Real-Time Budget Synchronization

  • • Eliminates stale financial views

Behavior-Based Budget Calculations

System adapts calculations based on user intent — supporting both spending limits and savings goals.

State-Aware Notification Engine

💸 Cost-Aware AI Processing System

Designed a selective AI execution model that minimizes operational cost while preserving full user functionality — ensuring intelligence is only applied when it provides real value.

Selective Intelligence Execution

AI processing is triggered only for meaningful changes, avoiding unnecessary computation and cost.

  • • Charges only on first-time or significant edits
  • • Skips minor or cosmetic updates
  • • Prevents redundant reprocessing

Incremental Processing Engine

Built a diff-based system that updates only changed financial entries instead of recomputing entire datasets.

  • • Stable entry IDs ensure consistency
  • • Detects edits vs new entries
  • • Minimizes unnecessary AI calls

Zero-Cost Recurring Automation

Auto notes reuse existing structure and pipeline logic, avoiding repeated AI classification on every run.

  • • No repeated AI calls for recurring data
  • • Scales without increasing cost linearly
  • • Maintains consistency with manual flow

Hybrid Free + Intelligent Usage Model

Users can freely create and manage notes, while AI-powered intelligence is applied selectively when needed.

  • • Non-financial notes incur zero cost
  • • AI activates only for structured extraction
  • • Encourages usage before monetization

Fairness & Trust-Oriented Billing

Designed a system that aligns cost with value, preventing users from being charged for redundant or low-impact actions.

  • • No charge for repeated or unchanged actions
  • • Eliminates hidden cost loops
  • • Builds long-term user trust and retention

This approach transforms AI from a constantly running cost into a controlled, value-driven system — making the product both scalable and economically sustainable.

Cross-User Financial Aggregation

System aggregates structured entries from all users’ notes into a unified reporting layer.
